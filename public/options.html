Github API Token

If you see API_LIMIT_REACHED alerts, you may want to consider adding a github API token. Read more about this in our docs, and pay special attention to the security considerations mentioned there (the token is stored unencrypted!).

To removed a previously added token, override it with a new token or no token (empty input field).

Cached Licenses

To keep API usage and workload low, licenseplate caches identified licenses and uses them for one hour. At the moment, you have (loading) licenses cached ((loading) of which are expired).

Expired licenses are automatically removed next time the cache is updated - this thus happens regularly while you are browsing github on non-ignored repositories.

Ignored Repositories
